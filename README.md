# ExamMind: Pro

**Интеллектуальный тренажёр для подготовки к техническим собеседованиям в IT**

---

## О проекте

ExamMind: Pro — это веб-приложение с интеграцией Telegram-бота, которое автоматизирует подготовку к техническим собеседованиям. Пользователь проходит собеседование с ИИ, включающее теоретические вопросы, алгоритмические задачи и блок soft skills. Нейросеть анализирует ответы, проверяет код, оценивает сложность решений и даёт персонализированные рекомендации.

---

## Основные возможности

- Выбор направления (Python, Java, JavaScript, системный дизайн, C++, Go) и уровня (Junior/Middle/Senior)
- Теоретические вопросы с оценкой полноты и логики ответа
- Алгоритмические задачи с редактором кода, запуском тестов и анализом сложности
- Soft skills с разбором по методике STAR
- Детальный отчёт с баллами и рекомендациями
- Личный кабинет с дашбордом прогресса и тепловой картой ошибок
- Telegram-бот для прохождения собеседований в диалоговом режиме
